## Usage

1. Start the server with `npm start`.
2. Access the API endpoints:

- `GET /items`: Fetch all items.
- `POST /items`: Add a new item.
- `PUT /items/:id`: Update an item by ID.
- `DELETE /items/:id`: Delete an item by ID.
